# Brew & Bean Coffee - Landing Page

## Project Overview

This project is a responsive landing page for a fictional coffee shop called Brew & Bean Coffee. It was created to demonstrate frontend development skills, including layout structuring, responsive design, and basic JavaScript functionality.

The goal of the project is to simulate a real-world commercial landing page with a clean interface and smooth user experience.

---

## Features

- Responsive design for desktop and mobile devices
- Structured landing page with multiple sections
- Hero section with call-to-action buttons
- Menu section with product cards and images
- Reviews section
- Contact form layout
- Smooth scrolling navigation using JavaScript
- Basic hover effects and UI interactions

---

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (DOM manipulation)
- Unsplash images
